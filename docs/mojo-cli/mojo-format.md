# mojo format

格式化 Mojo 源文件。

## Synopsis

`mojo format [options] <sources...>`

## Description

使用 Mojo 特定的 lint 工具格式化给定的一组 Mojo 源文件。

## Options

### Format options

`--line-length <INTEGER>`, `-l <INTEGER>`

设置最大字符行长度。默认值为 80。

### Diagnostic options

`--quiet`, `-q`

禁用非错误消息。

### Common options

`--help`, `-h`

显示帮助信息。
